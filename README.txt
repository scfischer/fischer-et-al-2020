README for Supplementary material of Sabine C. Fischer, Elena Corujo-Simón, Joaquín Lilao, Ernst H.K. Stelzer and 
Silvia Muñoz-Descalzo, "The transition from local to global patterns governs the differentiation of mouse blastocysts"

All code is provided as .nb and as .cdf. If a Mathematica licence is not available, the .cdf files can be used with the cdf-Player.
The cdf-Player can be obtained from https://www.wolfram.com/player/.

"DataFromMINS" contains the output of data I from MINS (Lou et al. Stem Cell Rep. 2014)
"EmbryoFeatures" contains the processed data I in Mathematica's .mx format
"EmbryoFeaturesSaiz" contains the processed data II in Mathematica's .mx format

All further information regarding the analysis is contained in the Mathematica notebooks.

"AllData" contains the processed data I-VIII in .json format for compatibility with other analysis software. 

"SupVideos" contains the three example image stacks relating to Sup. Info Fig 3.

For questions and comments please contact sabine.fischer@uni-wuerzburg.de
